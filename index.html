St. Louis Developer Connection

Homepage for St. Louis startups, developers, entrepreneurs, and investors

About

The purpose of this site is to be a homepage for the emerging startup scene in the Greater St. Louis Missouri metropolitan area.

If you would like to contribute to the site, for instance documenting an organization you are a member of or writing about what it's like to be a part of a startup, please contact @nill for instructions on how to get started.

If you are a designer, developer, or already familiar with how Jeykll works, pull requests are welcome!
